Installation

Instructions on how to manually install the extension via Chrome, Edge, and Firefox extension settings.

Usage

Press Ctrl+K (Windows) or Cmd+K (Mac) to open Spotlight. Type a Salesforce ID or command.

Supported Commands

  • home → Go to Salesforce Home
  • objects → Open Object Manager
  • account, contact, opportunity → Direct object navigation

FAQ

Q: Will this extension access my Salesforce data?
A: No. It only reads the URL of the current tab.
